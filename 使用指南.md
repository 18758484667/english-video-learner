# 英语学习视频工具 - 使用指南

##  快速开始(3步)

### 第1步: 安装Node.js(只需一次)

如果你还没有安装Node.js:
1. 访问 https://nodejs.org/
2. 下载 LTS (长期支持)版本
3. 双击安装包,一路点击"下一步"即可
4. 安装完成后,重启电脑

### 第2步: 启动程序

1. 打开文件夹 `english-video-learner`
2. **双击** `start.bat` 文件
3. 等待浏览器自动打开

首次运行会安装依赖,需要等待1-2分钟。以后每次启动只需几秒钟。

### 第3步: 开始学习

浏览器会自动打开 http://localhost:5173

#### 功能使用:

**1️⃣ 登录**
- 输入任意用户名(例如: "张三")
- 点击 "Start Learning"

**2️ 体验三层字幕**
- 点击顶部 "Video Player" 标签
- 点击绿色按钮 **"Load Sample"**
- 你会看到示例句子的三层字幕效果:
  ```
          狐狸     跳跃          ← 生词中文释义(大字,彩色)
  The quick brown fox jumps...  ← 英文字幕(白色大字)
  那只敏捷的棕色狐狸跳过了...   ← 整句翻译(灰色小字)
  ```

**3️⃣ 交互操作**
- 点击单词可以查看详情
- "Show/Hide Annotations" - 显示/隐藏生词释义
- "Show/Hide Translation" - 显示/隐藏整句翻译
- 底部按钮可以调整播放速度(0.5x - 1.5x)

**4️⃣ 英语水平测试**
- 点击 "Level Test" 标签
- 完成词汇量测试
- 系统会评估你的CEFR等级(A1-C2)

**5️ 生词本**
- 点击 "Vocabulary List" 标签
- 查看你收藏的所有生词
- 可以点击 "Export CSV" 导出

##  常见问题

### Q: 为什么浏览器是黑色的?
A: 这是深色主题设计,保护眼睛。所有文字都是白色的,可以正常阅读。

### Q: 如何停止程序?
A: 在命令行窗口按 `Ctrl+C`,然后关闭窗口即可。

### Q: 下次还要重新安装吗?
A: 不需要!第一次运行后,依赖已经安装好了。以后双击 `start.bat` 就能直接启动。

### Q: 可以上传自己的视频吗?
A: 当前版本是演示版,只支持加载示例字幕。完整视频处理功能将在后续版本添加。

### Q: 数据会保存吗?
A: 当前版本数据存储在浏览器中,关闭浏览器后会丢失。后续版本会添加数据库持久化。

## 🎯 核心功能说明

### 三层字幕显示(最重要!)

这是本工具的核心创新:

```
──────────────────────────────────────┐
│     [生词释义]    [生词释义]         │ ← 第1层: 生词中文(大字)
│  The quick brown fox jumps over...   │ ← 第2层: 英文原句(主显示)
│  那只敏捷的棕色狐狸跳过了懒惰的狗    │ ← 第3层: 整句翻译(小字)
└──────────────────────────────────────┘
```

**设计理念:**
- 第1层: 只有你不认识的单词才显示中文释义,避免干扰
- 第2层: 完整的英文句子,让你学习地道表达
- 第3层: 整句翻译帮助理解上下文

**颜色标注:**
- 🔴 红色: 超出你水平2级以上的生词
-  黄色: 超出你水平1级的生词
- ⚪ 白色: 你已掌握的单词

## 🛠️ 技术说明(可选阅读)

### 技术架构
- **前端**: React 18 + TypeScript + Vite
- **样式**: TailwindCSS (现代化CSS框架)
- **状态管理**: Zustand (轻量级状态管理)
- **视频播放**: ReactPlayer
- **后端**: FastAPI + SQLite (可选,当前未启用)

### 为什么选择这个架构?

**React的优势:**
- 组件化开发,代码清晰易维护
- 丰富的生态系统,功能扩展方便
- 性能优秀,用户体验流畅

**TailwindCSS的优势:**
- 无需写CSS文件,直接在HTML中写样式类
- 响应式设计,自动适配手机和电脑
- 现代化的UI组件库

**Zustand的优势:**
- 比Redux简单10倍
- 代码量少,易于理解
- 性能优秀
