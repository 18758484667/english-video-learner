# 后端配置指南

## 📋 前置要求

### 1. Python环境
- **版本**: Python 3.9 或更高版本
- **下载**: https://www.python.org/downloads/
- **安装时勾选**: "Add Python to PATH"

### 2. FFmpeg (视频处理必需)
- **Windows**: 
  - 下载: https://github.com/BtbN/FFmpeg-Builds/releases
  - 解压后将 `ffmpeg.exe` 放到系统PATH中(如 `C:\Windows`)
  
- **验证安装**:
  ```bash
  ffmpeg -version
  ```

## 🚀 快速启动

### 方法1: 双击启动脚本(推荐)
1. 打开 `backend` 文件夹
2. 双击 `start-backend.bat`
3. 等待服务器启动
4. 访问 http://localhost:8000/docs 查看API文档

### 方法2: 命令行启动
```bash
cd backend

# 首次运行需要创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 初始化数据库
cd app
python init_db.py
cd ..

# 启动服务器
uvicorn app.main:app --reload
```

## ⚙️ 配置百度翻译API

### 1. 申请API密钥
1. 访问: https://fanyi-api.baidu.com/
2. 注册账号并登录
3. 进入"管理控制台" → "开发者信息"
4. 复制 APP ID 和 密钥

### 2. 配置环境变量
在 `backend` 目录下创建 `.env` 文件:

```env
# 百度翻译API配置
BAIDU_TRANSLATE_APPID=你的APP_ID
BAIDU_TRANSLATE_SECRET_KEY=你的密钥

# Whisper模型配置(可选)
WHISPER_MODEL_SIZE=small  # tiny, base, small, medium, large
WHISPER_DEVICE=auto       # auto, cpu, cuda
```

**注意**: 如果不配置,系统会使用模拟翻译数据。

## 🔧 常见问题

### Q1: 提示 "ModuleNotFoundError: No module named 'xxx'"
**解决**: 重新安装依赖
```bash
pip install -r requirements.txt
```

### Q2: FFmpeg找不到
**解决**: 
1. 确认已下载FFmpeg
2. 将 `ffmpeg.exe` 复制到 `C:\Windows` 目录
3. 重启命令行

### Q3: 端口8000已被占用
**解决**: 修改启动命令的端口号
```bash
uvicorn app.main:app --reload --port 8001
```

### Q4: 视频处理很慢
**原因**: Whisper转录需要GPU加速
**解决**: 
- 使用较小的模型(tiny或base)
- 或等待CPU处理完成(较慢但可用)

## 📊 API端点

### 视频相关
- `POST /api/videos/upload/` - 上传视频
- `GET /api/videos/{id}` - 获取处理状态
- `GET /api/videos/user/{user_id}` - 获取用户的视频列表

### 用户相关
- `POST /api/users/` - 创建用户
- `GET /api/users/{id}` - 获取用户信息

### 词汇相关
- `POST /api/vocabulary/items/` - 添加生词
- `GET /api/vocabulary/items/{user_id}` - 获取生词本

### 测试相关
- `POST /api/tests/start/` - 开始测试
- `POST /api/tests/submit/` - 提交测试
