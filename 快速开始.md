# 英语学习视频工具 - 快速开始指南

## 🎯 第一步: 检查环境

### 必需软件

1. **Node.js** (前端运行)
   - 检查是否安装: 命令行输入 `node --version`
   - 下载地址: https://nodejs.org/

2. **Python** (后端运行)
   - 检查是否安装: 命令行输入 `python --version`
   - 需要版本: 3.9 或更高
   - 下载地址: https://www.python.org/downloads/

3. **FFmpeg** (视频处理必需)
   - 检查是否安装: 命令行输入 `ffmpeg -version`
   - Windows下载: https://github.com/BtbN/FFmpeg-Builds/releases
   - 安装方法: 解压后将 `ffmpeg.exe` 复制到 `C:\Windows` 目录

---

## 🚀 第二步: 启动应用

### 方法1: 一键启动(推荐)

直接双击项目根目录的 **`start.bat`** 文件

等待两个窗口都显示"成功"信息后,浏览器会自动打开应用。

### 方法2: 手动启动

#### 启动后端

```bash
cd backend
双击 start-backend.bat
```

看到以下信息表示成功:
```
========================================
  后端服务器启动中...
  API地址: http://localhost:8000
  API文档: http://localhost:8000/docs
========================================
```

#### 启动前端

打开新的命令行窗口:
```bash
cd frontend
npm run dev
```

看到以下信息表示成功:
```
VITE ready in xxx ms
Local: http://localhost:5173/
```

---

## 📖 第三步: 开始使用

### 1. 登录

- 打开浏览器访问: http://localhost:5173
- 输入任意用户名(如: "张三")
- 点击"Login"按钮

### 2. 英语水平测试(可选)

- 点击顶部导航栏的 "Level Test"
- 完成10道词汇题
- 系统会评估你的CEFR等级(A1-C2)

### 3. 上传视频学习

- 点击 "Upload Video" 标签页
- 点击 "Choose File" 选择本地视频文件
- 等待后台处理(通常需要1-5分钟,取决于视频长度)
- 处理完成后自动跳转到播放器

### 4. 观看视频

三层字幕显示:
- **顶层**(红色/黄色): 生词中文释义
- **中层**(白色大字): 英文原句
- **底层**(灰色小字): 整句中文翻译

交互功能:
- 点击任意单词查看详细信息
- 使用控制按钮切换字幕显示
- 调整播放速度(0.5x - 2.0x)

### 5. 查看生词本

- 点击 "Vocabulary List" 标签页
- 查看所有标注的生词
- 可以导出为CSV文件

## ⚙️ 可选配置: 百度翻译API

如果不配置,系统会使用模拟翻译数据(效果较差)。

### 申请步骤

1. 访问: https://fanyi-api.baidu.com/
2. 注册账号并登录
3. 进入"管理控制台" → "开发者信息"
4. 复制 APP ID 和 密钥

### 配置方法

在 `backend` 目录创建 `.env` 文件,内容如下:

```env
BAIDU_TRANSLATE_APPID=你的APP_ID
BAIDU_TRANSLATE_SECRET_KEY=你的密钥

# Whisper模型配置(可选)
WHISPER_MODEL_SIZE=small  # tiny, base, small, medium, large
WHISPER_DEVICE=auto       # auto, cpu, cuda
```

保存后重启后端服务器。

## ❓ 遇到问题?

### 问题1: 双击start.bat没反应

**检查**:
- 是否安装了Node.js和Python
- 命令行输入 `node --version` 和 `python --version` 确认

### 问题2: 上传视频后一直"处理中"

**可能原因**:
- FFmpeg未安装
- 后端服务器未启动
- 视频文件太大

**解决**:
1. 确认FFmpeg安装: `ffmpeg -version`
2. 查看后端控制台日志
3. 尝试上传较小的视频文件测试

### 问题3: 样式加载不正常

**解决**:
```bash
cd frontend
npm install
npm run dev
```
