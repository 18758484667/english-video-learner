# 英语学习视频工具 - 快速启动

## 🚀 三步启动(超简单!)

### 第1步: 找到项目文件夹
打开这个文件夹:
```
D:\Documents\Lingma Files\english-video-learner\
```

### 第2步: 双击启动
在这个文件夹里,找到并**双击**这个文件:
```
start.bat
```

### 第3步: 浏览器自动打开
等待几秒钟,浏览器会自动打开网站!

---

## ✨ 开始使用

### 首次使用:
1. 输入任意用户名(比如你的名字)
2. 点击 "Start Learning"
3. 点击顶部 "Video Player" 标签
4. 点击绿色按钮 **"Load Sample"**
5. 看三层字幕效果!

### 核心功能:
- **三层字幕**: 生词释义 + 英文原句 + 整句翻译
- **词汇测试**: 评估你的英语水平(A1-C2)
- **生词本**: 收藏学习中遇到的生词

---

## ❓ 遇到问题?

**问题1: 浏览器没自动打开**
- 手动在浏览器地址栏输入: `http://localhost:5173`

**问题2: 页面显示不正常**
- 按 Ctrl+Shift+R 强制刷新

**问题3: 想停止程序**
- 在黑色命令行窗口按 Ctrl+C
